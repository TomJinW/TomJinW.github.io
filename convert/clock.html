宝可梦金银 韩版汉化版/韩版，欧美版金银水晶时钟重置工具


关于时钟重置：在下方表格中输入中文/韩文训练家名称，零花钱，和训练家ID，重置密码会自动生成。

生成的密码适用于：

- 韩版，韩版汉化版：金、银
- 欧美版：金、银、水晶
- CKN&口袋群星&DMG汉化版：水晶，请前往这里重置

如何打开重置时钟界面：

此操作适用于欧美版金、银：

- 标题画面中按下 ↓ + SELECT + B

此操作适用于韩版/韩版汉化金银，以及欧美版水晶：

- 标题画面中按住 ↓ + SELECT + B
- 松开 ↓ + B 但是持续按住 SELECT
- 按住 ← + ↑
- 松开 SELECT
